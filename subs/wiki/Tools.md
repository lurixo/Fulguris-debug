# Tools

The tools we use to develop and test Fulguris:

## Testing & Debugging

- [App Manager](/Slion/Fulguris/wiki/App-Manager) - Comprehensive Android package manager for debugging and inspecting Fulguris
- [BrowserLeaks](/Slion/Fulguris/wiki/browserleaks.com) - Web-based testing tool for validating browser features and privacy
- [ADB](/Slion/Fulguris/wiki/ADB) - Android Debug Bridge for device communication and debugging
- [WSA](/Slion/Fulguris/wiki/WSA) - Windows Subsystem for Android for testing on Windows 11

## Development

- [Android Studio](/Slion/Fulguris/wiki/Android-Studio) - Official IDE for Android development
- [Gradlew](/Slion/Fulguris/wiki/Gradlew) - Gradle Wrapper for building and managing the project
- [SVG Path Editor](/Slion/Fulguris/wiki/SVG-Path-Editor) - Web-based tool for quickly modifying icon paths
- [Affinity Designer](/Slion/Fulguris/wiki/Affinity-Designer) - Professional vector graphics editor
